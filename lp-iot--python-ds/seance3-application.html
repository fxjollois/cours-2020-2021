Nous allons rgulirement utiliser ce code, donc nous allons le stocker dans une fonction nomme dessin(). Dans celle-ci, nous allons mettre en paramtre les $x_j$ et les $y_j$, le chiffre, ainsi qu'un graphique dans lequel nous allons mettre le dessin. Ceci nous sera utile pour faire plusieurs reprsentations de chiffres.

In [7]:
def dessin(p, x, y, chiffre):
    p.plot(x, y)
    p.set_title("Chiffre : " + str(chiffre))
    p.axis("off")
    p.set_xlim([-1, 101])
    p.set_ylim([-1, 101])

fig, ax = plt.subplots()
dessin(ax, x, y, chiffre)

Ensuite, nous crons une liste de DataFrame, un pour chaque chiffre. La fonction query() permet donc de slectionner des lignes d'un DataFrame en fonction d'une condition (ici, chiffre gal 0, 1, ..., 9). Pour viter les problmes d'index plus tard, nous devons les rinitialiser pour chaque DataFrame, avec la fonction reset_index(), en mettant drop vrai. Ceci permet d'oublier les numros de ligne du DataFrame global et que ceux-ci recommencent de 0 pour chaque sous-ensemble.
